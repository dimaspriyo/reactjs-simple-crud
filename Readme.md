## Prerequisites
- Port 3000 - 3002
- docker-compose

## Getting Started

 1. Clone [https://github.com/dimaspriyo/reactjs-simple-crud](https://github.com/dimaspriyo/reactjs-simple-crud)
 2. Run **docker-compose up -d**
 3. Check deployed container status 
 **docker ps**
 4. Browse to http://localhost:3000
